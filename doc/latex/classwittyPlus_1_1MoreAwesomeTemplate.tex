creates a Widget and it's field and binds it into the template. 

The template must be of the format:

$<$message id=\char`\"{}some-\/template\char`\"{}$>$  \$\{name-\/label\} \$\{name-\/field\}  $<$/message$>$ $<$message id=\char`\"{}name-\/label\char`\"{}$>$Name$<$/message$>$ $<$message id=\char`\"{}Name\char`\"{}$>$Name$<$/message$>$

In this case 'name' would be the prefix. The label would be labeled: tr(\char`\"{}name-\/label\char`\"{}) If no tr(\char`\"{}name-\/label\char`\"{}) is found .. it'll use tr(\char`\"{}Name\char`\"{}) (with an uppercase first letter)
