Detailed Description

The standard Wt::WValidator seems to make the assumption that nearly all validation that requires showing a message will be done on the client side. This class extends Wt::WValidator to provide a 'validateWithMessage' function on the server side. It is intended to be used with validation that needs to be done server side, but where a message must be shown to the client also. We don't show the message though, we just provide it in a signal when validate is called
