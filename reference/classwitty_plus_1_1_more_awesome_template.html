creates a Widget and it's field and binds it into the template.

The template must be of the format:

<message id="some-template">

${name-label}
${name-field}

</message> <message id="name-label">Name</message> <message id="Name">Name</message>

In this case 'name' would be the prefix. The label would be labeled: tr("name-label") If no tr("name-label") is found .. it'll use tr("Name") (with an uppercase first letter)

Template Parameters:
TThe type of the widget eg Wt::WLineEdit
Parameters:
labelA pointer to hold the label.
widgetA pointer to hold the WFormWidget
prefixThe prefix to do all the naming stuff.
